## Features That Will Help You Find the Perfect Font

*   **Huge Font Library:** Preview any font from the Google Fonts library.
*   **Customizable Properties:** Adjust the font size, weight, and style to see how it will look in your project.
*   **Compare Fonts:** Compare multiple fonts side-by-side to find the one that's right for you.
*   **Generate CSS:** Get the CSS code you need to use the font in your project with a single click.
*   **Accessibility Checks:** We'll check the readability of your chosen font to ensure it's accessible to people with visual impairments.

## How to Use the Font Preview Tool

1.  **Choose a Font:** Browse the Google Fonts library and select the font you want to preview.
2.  **Customize the Properties:** Use the controls to adjust the font size, weight, and style.
3.  **Compare Fonts:** Add multiple fonts to the comparison view to see them side-by-side.
4.  **Get the CSS:** Click the "Get CSS" button to get the CSS code you need to use the font in your project.
