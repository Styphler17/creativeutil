## Introduction

Web accessibility is not just a good idea, it's a legal requirement in many countries. But more importantly, it's the right thing to do. An accessible website is one that can be used by everyone, regardless of their abilities.

But how do you know if your website is accessible? That's where our **Free Accessibility Checker** comes in. It's a simple tool that helps you test your HTML code or live website for common accessibility issues.

## Features That Help You Build a More Inclusive Web

*   **Check HTML Code or Live Websites:** Paste your HTML code or enter a URL to check for accessibility issues.
*   **Comprehensive Checks:** We check for a wide range of accessibility issues, including:
    *   **Color Contrast:** We'll check the contrast ratio of your colors to ensure they meet WCAG guidelines.
    *   **Alt Text:** We'll check your images to make sure they have meaningful alt text.
    *   **Heading Structure:** We'll check your heading structure to make sure it's logical and easy to follow.
    *   **And more!**
*   **Actionable Feedback:** We'll provide you with clear, actionable feedback on how to fix any accessibility issues we find.

## How to Use the Accessibility Checker

1.  **Choose Your Input:** Choose whether you want to check HTML code or a live website.
2.  **Enter Your Input:** Paste your HTML code or enter the URL of the website you want to check.
3.  **See the Results:** We'll show you a list of any accessibility issues we find, along with suggestions on how to fix them.
